# PostIt

临时文本暂时存储。HTML和MD格式。小窗模式。Electron软件窗口模式。

## 设计

### 外观

- 简约
- 跳转条
- 展示模式
	- 不可编辑，设置默认隐藏
	- 设置仍然可以在页面右下角打开
	- url参数
	- 读取config
- ~~点击保存才能保存（这是特性）~~
- 定时保存上传
- 上面编辑/展示，下面是设置。

### 数据

- 保存在{文件储存}/postit/{name}/content，{文件储存}/postit/{name}/config.json
- content直接保存文本内容
- config保存：
	- 文本格式
	- 修改时间
	- 作者
	- 展示模式
	- 最大宽度
